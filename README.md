## Getting Started

This project require NodeJS v12.

# Install
Clone this project

```bash
> git clone https://github.com/ZeusFtrOfc/whatsapp-bot.git
> cd whatsapp-bot
```

Install the dependencies:

```bash
> npm install 
```

### Usage
Run the Whatsapp bot

```bash
> npm start
```

after running it you need to scan the qr
